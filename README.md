# easyeda-api-skill

嘉立创EDA专业版AI SKILL；EasyEDA Pro Skill for AI

你可以使用 AI 编程工具（OpenCode、QwenCode、KimiCode 等）加载该SKILL进行快速开发嘉立创EDA专业版的扩展插件，配合Run API Gateway扩展、extension-dev-skill、extension-dev-mcp-tools可以更方便的进行扩展的开发。

使用方法：
1、下载本仓库在本地
2、解压后放入编程工具的SKILL目录下即可


You can use AI programming tools (such as OpenCode, QwenCode, KimiCode, etc.) to load this SKILL for rapid development of the extension plugin for EasyEDA Pro. By combining it with the [Run API Gateway](https://github.com/easyeda/eext-run-api-gateway) extension, [extension-dev-skill](https://github.com/easyeda/extension-dev-skill), and [extension-dev-mcp-tools](https://github.com/easyeda/extension-dev-mcp-tools), you can more easily develop extensions.

Usage:
1. Download this repository locally
2. After decompression, simply place it in the SKILL directory of your programming tool
